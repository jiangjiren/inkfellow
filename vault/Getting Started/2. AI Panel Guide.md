# AI 面板使用指南

Notes App 内置的 AI 面板基于 **LLM Wiki** 理念设计：让 AI 读懂你的笔记，随时回答你的问题。

## 支持的 AI 服务商

| 服务商 | 接入方式 | 推荐程度 |
|--------|----------|----------|
| **Claude 官方** | Anthropic 账号登录，无需 API Key | ⭐⭐⭐ 最推荐 |
| **DeepSeek** | 填入 API Key | ⭐⭐⭐ 性价比高 |
| **OpenRouter** | 填入 API Key，可选 200+ 模型 | ⭐⭐ 模型最全 |
| **自定义** | 任意兼容 Anthropic API 的服务 | ⭐⭐ 灵活 |

## 核心用法：划选引用

1. 在笔记中用鼠标选中任意文字
2. AI 输入框会自动出现该内容的引用
3. 直接追问，无需手动复制

**示例**：选中上面表格中的"DeepSeek"这行，然后问 AI：「这个服务商怎么申请 API Key？」

## 会话持久化

对话历史会自动保存，刷新页面或关闭浏览器后重新打开，之前的对话仍然在。

点击对话框顶部的 **🗑 清除** 按钮可以开始新对话。

## 切换模型

点击 AI 面板右上角的设置图标，可以：

- 添加多个服务商账号
- 在不同服务商之间一键切换
- 为不同任务选择不同的模型（快速回答用小模型，深度分析用大模型）

# 快速上手

## 第一步：准备你的笔记文件夹

Notes App 支持任何包含 `.md` 文件的文件夹，与 [Obsidian](https://obsidian.md) 格式完全兼容。

```
my-vault/
├── 工作笔记/
│   ├── 项目 A.md
│   └── 会议记录.md
├── 读书笔记/
│   └── 原则.md
└── 日记/
    └── 2025-01.md
```

## 第二步：配置环境变量

复制 `.env.example` 为 `.env.local`，填写以下必要配置：

| 变量 | 说明 |
|------|------|
| `VAULT_PATH` | 你的笔记文件夹绝对路径 |
| `NOTES_BASIC_AUTH_USERNAME` | 登录用户名（自己设置） |
| `NOTES_BASIC_AUTH_PASSWORD` | 登录密码（自己设置，不能为空） |
| `SITE_URL` | 你的域名或公网 IP |

## 第三步：构建并启动

```bash
npm run build
npm start
```

打开浏览器访问 `/notes`，输入你设置的用户名和密码即可登录。

## 登录账号说明

> ⚠️ 没有注册流程。登录账号就是你在 `.env.local` 里填写的用户名和密码，修改后重启服务生效。

## 可选：启用 AI 面板

```bash
cd claude-chat
npm install
npm start   # 在 8082 端口启动 AI 服务
```

然后在 Nginx 中把 `/notes-claude/` 转发到 `127.0.0.1:8082`。
